# spider_projects
python 爬虫小demo，后续会继续更新
 - 1.全民K歌歌曲爬取
 - 1.1 操作步骤：
 -（1）克隆项目到本地，打开python项目，创建存放音乐的文件夹
 -（2）通过全民k歌分享歌曲（qq、微信、朋友圈都可，仅需获得分享地址）
 -（3）运行项目，输入分享地址，歌曲即可下载
 - 1.2 博客地址：
 - https://blog.csdn.net/Twinkle_sone/article/details/106866925
